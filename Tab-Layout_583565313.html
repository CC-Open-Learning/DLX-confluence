VARLab : Tab Layout

Created by Carolina Naoum Junqueira, last modified on Dec 20, 2022

The tab layout was created for simulation in 2D with intuitive, non-linear navigation. The user can navigate the tabs by clicking on them in any order, using the arrows at the bottom of the panels or using the left and right arrow keys on the keyboard.

The panels are used to display the learning material and can be done with the use of images, videos, text, and animations.

There are currently two tabbed layout prefabs:

  1. Horizontal Tab Layout - Topic tabs are arranged horizontally above the panels.

  1. Verticle + Horizontal Tab Layout - Topic tabs are arranged vertically on the left of the panels and the subtopic tabs are placed horizontally above the panels.
