Engage Hydraulics (UP)

Jacob Nelson (Unlicensed)

Key Features:

  • The ability to press the hydraulic engage button

    • Showcase button highlight

  • When button depressed and held the arms move upward

    • When button released the arms stop in place

  • The ability to move the arms completely upwards

Release Hydraulics (DOWN)

Jacob Nelson (Unlicensed)

Key Features:

  • The ability to press the hydraulic release lever

    • Showcase lever highlight

  • When lever depressed and held the arms move downward

    • When lever released the arms stop in place

  • The ability to move the arms downward to the ground

Engage Locks

Jacob Nelson (Unlicensed)

Key Features:

  • The ability to press on and interact with the lock lever

    • Showcase lever highlight

    • Showcase how the locks should be engaged when lever is positioned at 90 degrees

  • Touch on how the functionality of this lever is currently not complete (but don’t focus on it).

Height Cut-Off Bar

Jacob Nelson (Unlicensed)

Key Features:

  • Using debugging keyboard controls, raise the arms upwards and click on the cut-off bar to show that it is functional

    • Showcase that when the arms move downwards clicking the bar does not stop movement
