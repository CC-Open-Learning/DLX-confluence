VARLab : CCSIF Trucking Starting Broken States

Created by User 763fb, last modified on Nov 19, 2024

Below are the possible inspections that will start broken during a module B play-through, including both inspections.
This list includes all unique inspections that will uniquely appear in the first and second trucking inspections. It also includes linked inspections which include broken states that will always appear in the first inspection, with one to many linked broken inspections that will start broken in the second inspection if the original broken state isnâ€™t fixed in the first. Lastly it includes bad road choice inspections which are broken inspections that will appear if the player chooses that bad road choice in between their first and second inspection.

Possible starting broken inspections are broken up by the point of interest at which they are present. They are then group by how many of the group are able to show up (e.g. 1 broken inspection Between the multiple listed to the right will randomly start broken at the start of the inspection).
