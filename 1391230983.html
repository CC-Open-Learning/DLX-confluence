Seek and Find Beginner Mode

 Testing the functionality specific to Beginner mode:

  • No time limit.

  • Infinite hints.

    • VALIDATE all of the hints are correct and useful.

    • Can hints show the names of other bags/cabinets/compartments? Should the hints only reference other items rather than locations?

    • Do hints toggle the hover effect for the icon and background separately?

    • Are hints disabled when that explorable is found?

  • No personal best system.

  • Resets if the player aborts (via home or quit) without completing the list.

  • Returning to the main menu will prompt the user with a confirmation dialog that reminds them their progress will NOT be saved.

Seek and Find Advanced

Testing the functionality specific to Advanced mode:

  • Specified time limit that starts after player clicks “Let’s go”

  • No hints are available.

  • Personal best time is recorded ONLY if the player submits a perfect list for the scenario and improves on the previous best time (or sets a first time).

  • Can remove items from the list (and re-add then re-remove).

  • Can’t add the same item twice from the list.

  • Can add acceptable alternatives to the list.

  • The system accepts alternatives as specified by the CE.

    • Validate each of these alternatives: Ambulance Seek & Find - Scenario Details.docx

    • If an alternative is added to the list, a successful completion should list the alternative and not the “primary” explorable answer

  • Can set a new personal best with an alternative list from the previous successful submission.

  • Can submit an incomplete list and receive feedback.

  • Feedback shows Correct, Incorrect, and Not Found results.

  • Returning to the main menu will prompt the user with a confirmation dialog that reminds them their progress will NOT be saved

OUT OF SCOPE

Security Testing - No sensitive data is stored or transmitted. No grade items are created and no evaluation is performed in the sim. Exception: Personal best scores as outlined above.

Anything related to Compatibility for Screen Readers for accessability.

Headsets - This sim is not planned to be played in a headset like a MetaQuest 3.
