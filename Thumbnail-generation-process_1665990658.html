VARLab : Thumbnail generation process

Created by User 55f04 on Mar 06, 2025

Overview

This document encompasses the process of generating thumbnails for in-game objects used in Trades Electrical. It will specifically focus on generating thumbnails for Device Mountables but this can generally be applied to any object that would possibly be added in the future.

Collecting image snapshot

The first step of the process is to find the object that you are looking to generate the thumbnail for, in this case a device.

I would go to the prefab view of the object so I can get a closer look and adjust the camera angle for consistent snapshots for all devices. For this example, I am going to use the Light Switch Single Pole prefab. Go to Assets/Prefab/Mounting/Devices/Light Switches and double click the object to open prefab view.

Notice the background color of the prefab view above. You will need to set the background to a color that contrasts well with the object so that the image can be isolated from the background cleanly (a crucial step later in the process).

To set the background color, head over to the Edit section of the toolbar in Unity editor and click on Preferences. Then click on Colors and find Background for Prefabs.

Make sure to maximize the scene as shown above to get better resolution/details for the object you are taking a snapshot of. Adjust the camera angle and take a screenshot of the prefab as shown below (Ensure the size of the screenshot shown in red, is similar if not the same).

Editing the image

Once the image snapshot is saved, you will need to edit the image such that the background is removed and the size of the image is within certain constraints that is consistent with other thumbnails.

You can use any application to get this done but I personally use Paint.Net because it is the easiest way to do so.

Open the image with Paint.Net and select the Magic Wand as shown below:

Reduce the Tolerance of setting to zero and click on the background of the image and hit the delete button to remove any background color “areas” from the object as shown below:

Once the background has been removed, you will need to use the Rectangle Selection Tool to target only the object for resizing. The selection proportion for the object in the image should be similar if not the same as shown below.

Once selected, head over to the Image option in the toolbar and select Canvas Size. Ensure the Width of the canvas matches the height as shown below

Once complete, you will get a square image of the object. This would need resizing as well to fit within constraints for the thumbnails and for consistency with other thumbnails. To do this, simply head over to the Image option in the toolbar and select Resize. Here you will ensure the image is 512x512 for Width and Height of the image as shown below:

Save this image as this would then be used as thumbnail in the respective UI in the sim.

Loading thumbnail into Unity

For Trades Electrical, the thumbnails folder is in the path Assets/Art/Sprites/UI/Thumbnails. Under this folder you can place your thumbnail into the respective folder; For devices the folder for device thumbnails is Device Options.

Once the thumbnail has been placed in its folder. Set the thumbnail settings in the inspector as shown below:

Once set, you can now use this thumbnail in a Visual Element expecting an image in UI toolkit.
