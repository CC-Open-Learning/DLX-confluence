VARLab : File Naming Conventions

Created by Hannah Schmitting (Unlicensed) on Jun 24, 2021

File names will be split into sections, each separated by an underscore. Each section will follow a lowercase-uppercase format (with the exception of prefixes and campus labels), ex. “wallTiles”, “floorTiles”, “wingA”. Section order: Prefix, Campus Label, Description, Wing.

Ex.

DOON_wallTiles_wingA.png

NA_DOON_floorTiles_wingE.png

Prefixes:

NA = Not Applicable. This file is no longer in use and has been archived for posterity.

Campus Labels:

DOON = Kitchener, Doon

WALOO = Waterloo

CAM = Cambridge

GPH = Guelph

BTFD = Brantford

UG = University Gates
