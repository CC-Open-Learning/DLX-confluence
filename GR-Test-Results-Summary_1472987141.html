Downloaded Results - Max download size

Description

Create an inspection log that includes 1 photo of every inspectable item in the entire DLX (total 354 photos). End Inspection and review the summary, then download results.

  • Confirm that all 354 photos are downloaded. Each zip archive will contain up to 150 image files, so there will be multiple zip files. The total number of photos in all zip files must add up to 354.

  • Confirm that the 354 photos are correct - there are no duplicate files (ie. you did not get the same first 150 images in every zip)

  • Repeat test with several different popular web browsers, such as Chrome, Edge, and Firefox

Downloaded Results - download failure - disk space

Description

Confirm that the download will fail gracefully when there is not enough disk space for the download.

Configure your web browser to save to a very space-restricted location on your local machine (e.g. thumb drive)

End Inspection and proceed to download results and photos - report what happens.

NOTE: This test is designed to fail - it will not be possible to download results when inadequate disk space is available. The purpose of this test is to document what happens in these circumstances.

Downloaded Results - download failure - network connectivity issues

Description

Confirm that the download will fail gracefully when there is an interruption in network connectivity during the download.

Right after starting your results download, interrupt your internet connection. You could do this by toggling the wifi setting for your device, causing it to lose/regain/lose network connection. Try different timing intervals, to see how long you can be offline before the dowload will fail. Try different web browsers.
