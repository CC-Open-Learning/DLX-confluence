Before You Begin

This page covers how to implement the UI based and in-game tooltips in PHI. This workflow assumes that you have an understanding of UI Builder and the PHI scripts, and is up to date as of 08/14/24. Steps follow the default workflow and include examples that could be slightly different depending on the object.

This process assumes you have setup the button in the UI already

Adding a UI Tooltip

Step 1: Update the UI Document Script

First needed is the declaration of the label for the tooltip. Here it is called _exampleTip;

If you havenâ€™t already, you can also declare the Button here.

Next the SetupTooltips() function needs to be updated with two callbacks. The callback with the Flex DisplayStyle is for when the tooltip is visible and with the DisplayStyle None, it is not visible. Here it is using the already existing button _exampleBtn, but you should have created your own.

Step 2: Create the Label in UI Builder

Open the UI where the tooltip will be added.

Now while in the UI Builder, it will be showing the existing UI buttons and tooltips in PHI.

Now find the label under controls and drag it into the hierarchy below the button that will be using the tooltip.

Step 3: Setup the Label In UI Builder

First, under attributes only the text option needs to be changed to what the tooltip will say.

Second, under Display, the display option should be turned off as the tooltip should not be visible to start.

Next, under position, the position mode should be set to Absolute. For the anchors, these will differ based on where the tooltip should be placed. Each of the side values determine how far the label will be displayed from the side being set.

Next, under Spacing, the padding should be changed to be 10px on the left and right sides.

Under the Text formatting, the Color RGB should be set to 255, 255, 255, 255 to be a white color.

The background color should be set to 30, 72, 119, 255 to be a nice blue like the other tooltips.

Finally, for the border, the radius should be set to 8px.

Step 4: Setup the tooltip settings

The Object Tooltip script includes two different settings to consider when setting it up.

The first setting is the Tooltip text that will be displayed on the tooltip for the player to read.

The second setting relates to the POI from which the tooltip can be viewed. This setting is more important for objects like the movement tiles that can be seen from multiple POIs.

Click the drop down arrow to display the array or POIs, then select the plus symbol on the bottom right of the list to add a POI.

To add the POI find it in the hierarchy on the left side and drag it onto the element that was just created.

Additionally, to add multiple POIs, the number of POIs can be added on the right side of the list.
