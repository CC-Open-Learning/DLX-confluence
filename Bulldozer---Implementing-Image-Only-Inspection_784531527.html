VARLab : Bulldozer - Implementing Image-Only Inspection

Created by Armando Interaminense on Aug 16, 2023

This article will show how to implement the Image-Only Inspection in the Heavy Equipment DLX.

Â Instructions

  1. Add the Image-Only Inspection component to the game object that will do the Image-Only Inspection and fill in the fields properly.

  2. Create the Image Display Window inside WindowManager/Windows. Add all the components in the image below, or copy a similar window, for example, the Video Player Window, and add the Image Player UI component to it. If you are duplicating the Video Player Window, delete the Video Player UI.

  3. Open Preferred Types Editor to add ImageOnlyInspection and ImagePlayerUI that will be used in the FSM.

  4. In the FSM prefab and a Blackboard variable of ImagePlayerUI type. Then serialize with the Window object you created before.

  5. In the FSM add a conditional statement that checks whether CurrentInspection is ImageOnlyInspection type and leads to the Repeatable Inspection Lane.

  6. In the subgraph Repeatable Inspection Lane create a node with that set Current UI to the Image UI and add the CheckType function to its connection to check whether it Inspection is of ImageOnlyInspection type.
