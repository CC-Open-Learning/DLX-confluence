VARLab : TE Animation Creation and Integration

Created by User 1c73e, last modified on Apr 19, 2024

This page will outline how to create new animations for the TE project and how to integrate these animations into new and existing controllers. This workflow assumes using the existing equipment locations.

 Instructions

  1. Navigate to Art > Animations > Assets

  2. Select one of the controllers with your desired location; for example, select round box ceiling if you are adding equipment that will be mounted on the ceiling. Copy and paste the controller, and rename.

  3. Select any of the animations, copy and paste, and rename

  4. Open your new controller by double-clicking. You should be presented with the following view

  5. Drag your new animation into each Set location. After doing so, when clicking on either Set 1 or Set 2, you will see the animation is assigned in the “Motion” slot in the right inspector:

  6. After following the steps outlined in the Equipment Integration Workflow TE Equipment Integration Workflow , select an equipment option from the main scene hierarchy, then change the controller in Animation component to your newly created controller:

At this point, you can change the animation of ANY option within your new equipment category, which will change for each location.
