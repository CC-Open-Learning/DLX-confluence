VARLab : Workflow for 3D Inspection

Created by Armando Interaminense on Apr 18, 2023

Creating a 3D inspection

Step 1: Creating a 3D inspection container instance

Go into the prefab editing mode by clicking on the rightmost arrow.

Find the mesh you require for your inspection. For this example, let’s consider the Sprocket DS GameObject shown below.

Step 2: Adding inspection elements

Add the following components and serialize the required fields appropriately.

You don’t need to serialize the Camera field in the prefab mode.

keep interactable empty

Step 3: Adding a new camera for this inspection

Go to Camera Positions prefab.

Duplicate any child object to create a new camera position for the new inspection.

  • instructions for setting up the camera view

  • Include adding a camera component to see how it looks but mention that they need to test because it isn't completely accurate

Go back to the scene and serialize the newly created camera inside the Multiple Choice Inspection element.

Step 4: Inserting the inspection into the Inspection Manifest

As the final step, click on the Bulldozer prefab and navigate to the Inspection Manifest component.

Click on Find Inspection Elements to insert the newly created 3D Inspection into the Inspection Order list.

Place it in the correct order and override changes to the Bulldozer prefab.

Congratulations! The 3D inspection you created is now ready to use. You can now go ahead and test it.
