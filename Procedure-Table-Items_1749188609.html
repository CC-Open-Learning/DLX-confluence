Add Procedure Table Items in Biopsy Procedure

About:

This page contains information on how to add items to the procedure table in the Biopsy Procedure.

Assumptions:

This guide assumes that the 3D model object of the item you want to add to the procedure table of the biopsy procedure is already imported as a project asset and that you will have the appropriate meshes and textures for that object item to appear as needed as an item on the table.

Resources:

Figma Latest Designs

Miro Class Diagram

Biopsy Task System

SharePoint RT Scripts

Preparations:

Check Figma, RT Scripts and other documentation for latest designs, notes and changes.

Process:

Open up prefab of Prefab_Table_Overbed_Prepped.

Drag and drop the 3d model object on to the poly_Table_Overbed_Top prefab object in the hierarchy window.

Fix positioning of item on table.

Add Box Collider, Outlinable, Interactable and Biopsy Table Item scripts to the item in the table prefab.

Link procedure equipment item to Biopsy Table Item script by selecting it in the drop down menu.

Adjust position and rotating of the object so it will look like it is laying on the procedure table.

View of item on table in scene.

Add the item to the ProcedureTablePOI.
