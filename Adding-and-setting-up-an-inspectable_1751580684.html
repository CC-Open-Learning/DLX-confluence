Before Beginning

Prerequisite: The art assets are already imported into Unity, the materials are assigned to the models, and the models have been converted into prefabs. It also assumes you're familiar with the basics of Unity's Prefab system.

If you need to import the assets please see this page: https://varlab-dev.atlassian.net/wiki/pages/resumedraft.action?draftId=1070006323&draftShareId=c7cab80c-7ccf-45bb-bd5b-39f0a55e734c

To learn about prefabs please see: https://learn.unity.com/tutorial/prefabs-e#5f49e5bcedbc2a280d7cf66f

Create A Basic Inspectable

Step 1: Adding the Prefab Inspectable

  • Open the prefab scene where the inspectable object is expected to belong. <Piercing Area for example>

Within the Assets/Resource/Prefab folder there is a prefab called InspectablePrefab

  • Drag this into the prefab scene mentioned above

  • Unpack the prefab by right click â†’ Prefab â†’ Unpack

  • Rename InspectablePrefab to the new inspectable should be called.

Step 2: Placing the Asset

  • All assets are located in Assets/Prefabs/Base Prefab. Find the asset need to be inspected

<I will drag the Tool chest into the scene for example>

DO NOT unpack Asset prefab

Step 3: Convert Asset into Inspectable Object

Now we have empty InspectablePrefab and an asset prefab

  • Take the transform of the asset object we placed by copying the component

  • Paste that into the empty InspectablePrefab

  • The location of the asset and InspectablePrefab is matched

  • Drag the asset under the empty compliant.

For assets that have both compliant and non-compliant state, find the name of non-compliant object which should be mentioned in the ticket, then drag into non-compliant

Step 4: Setting Up the Components

  • Adjust the Box Collider by clicking the icon below. Make sure the Box Collider fits the object from all angles.

  • Choose the Outlinable (if it is unchecked) and Add all basic so you can see highlighting interactable object.

  • Add Inspectable Object.cs as a component.

  • Follow the information in the Jira ticket about the inspectable Object : Name, Location.

  • Drag the Camera and State from Hierarchy into inspection object

Edge-case

Toggleable

This is a special case where some inspectable objects might have parts that can be toggledâ€”for example, a lid that needs to be opened or a piece that needs to move in order to be properly inspected. Taking the Trash Can as an example.

  • Simply, just add Toggleable Inspectable script and drag the Camera and TrashCan into right place

So when the inspectable object which is in the toggleable list is clicked on, It will toggle from on->off or off â†’ on. The Trash Can in the scene to be covered, but then when clicked to be inspected in the picture within the inspection window the lid will be opened.

Object Viewer Inspectable

Inspectable Object is displayed with a black background and the learner is able to rotate it and zoom in.

<Using Forceps as an example>

  • We do not need Camera for Object Viewer Inspectable, so Camera should be removed from InspectablePrefab

  • Copy Rotation and Scale of object

  • Under Assets/Scriptable Objects/ObjectTransformDetails create a new ObjectTransformDetails

  • Rename the new file

  • Paste Rotation and Scale (copied above and will be modify later) here

  • Go back to Inspectable object , add ObjectViewerInspectables script. RotateObject and WorldObject script will be automatically added

  • Drag new ObjectTransformDetails into Transform Details

  • Play the game and adjust the rotation and scale as needed.

Message Inspectable

A message is displayed under the picture in the inspection window, the message is what is used to identify if something is compliant or not.â€‹

Prerequisite: The VELCRO notification must be imported as a child of the Inspection Window.

<I will use Handwashing as example>

  • Create new Scriptable Object NotificationSO which will be located under Assets/Scriptable Objects/Inspectables. It should be renamed

  • Add MessageInspectable.cs into the Handwashing and drag the Scriptable Object NotificationSO.

Step 5 : Create Prefab for Inspectable Object

So far, we have the InspectablePrefab with all the necessary components

  • We want create a new Prefab of the combined object, to do this you will simply click the InspectablePrefab (should be renamed) from hierarchy and drag this into the according location within the folder structure.

It should be located in Assets/Prefabs/Inspectables

Now, the object is ready for inspection.
