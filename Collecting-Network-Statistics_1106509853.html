Setup

Testers can open a command terminal by typing powershell in the Windows Start Bar and running the Windows Powershell application:

Before Testing

Set a starting network traffic value with the command:

$startBytes = Get-NetAdapterStatistics -Name “Wi-Fi” | Select ReceivedBytes -ExpandProperty ReceivedBytes
$startTime = Get-Date -UFormat %s

During Testing

Record latency in the connection to eConestoga with 

ping conestoga.desire2learn.com

After Testing

Record final network volume with the command:

$endBytes = Get-NetAdapterStatistics -Name “Wi-Fi” | Select ReceivedBytes -ExpandProperty ReceivedBytes
$endTime = Get-Date -UFormat %s
echo (($endBytes - $startBytes)/($endTime - $startTime))
