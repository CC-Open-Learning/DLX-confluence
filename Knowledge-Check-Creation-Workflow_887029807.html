VARLab : Knowledge Check Creation Workflow

Created by User 5ebc2, last modified on Mar 04, 2024

This page will detail the workflow required in creating a new knowledge check for a lab. Specific steps will be outlined and accessible through the list below.

Creating a Knowledge Check Task Scriptable Object

The process of creating a standard knowledge check has been standardized to a point, although there are a few decisions that need to be made throughout that will determine the approach that you should take.

For now, we will just be creating the base scriptable object. Follow the steps below or use the image below.

  1. Navigate to \Scriptable Objects\Lab Tasks and enter the folder of the lab that you are creating a knowledge check for.

  2. Once you are in the folder for your selected lab, right click and create a new KnowledgeCheckTask under ScriptableObjects → Tasks → KnowledgeCheckTask

  3. With this created, it is time to assign the values. Proceed below

Populating Knowledge Check Task Values

Every standard knowledge check consists of:

  • A Question, (text and audio)

  • Feedback (text and audio)

  • Options (We will get into this later)

For this example, I will be using the Lab 1 knowledge check.

Question Text

To find any text related to knowledge checks (and all other text within the Sim), refer to the MPC Script on Sharepoint under the “Voiceover and Script” folder.

Once you find the associated Question text, copy and paste it into the Question Text field in the scriptable object that you created.

Question Audio

To find any audio related to knowledge checks (and all other audio within the Sim), refer to the Voice Clips folder on Sharepoint within the “Voiceover and Script” folder. Once you find the associated Question audio, download it and import it into Unity,

Create a new Audio Segment (\ScriptableObjects\AudioSegment) and add the audio clip to the Audio Clips list in the segment. Assign the completed Audio Segment in the task scriptable object that you created.

This process is the same for both the correct and incorrect feedback.

IsPrediction

This value determines whether or not this knowledge check is a prediction knowledge check. In a standard scenario, it will stay unchecked. Knowledge checks with a prediction will be covered elsewhere on this page.

With the basics done, next I will cover creating and assigning Knowledge Check Options

Populating Knowledge Check Option Values

Every Knowledge Check Option consists of:

  • An Image

  • Text that accompanies the image

  • Feedback text that appears in the feedback panel (where applicable)

  • Checkbox for if the option is the correct answer (where applicable)

As of now, the maximum amount of options in a knowledge check is 4. Clicking on any of them will enable a designated outline (either green, or red) depending on whether the answer is correct, or incorrect respectively. A new option can be created by clicking the + icon on the options list

Image

Images for the knowledge checks can be found on Sharepoint under Assets\2D\MPC-KC Images, however, most images you would need will likely already be in the project under Images\KnowledgeCheckImages

To decide what image to use, refer to Labs 6-10 Lab Procedures and find the knowledge check step in the table related to the lab you are working on. The text associated with the option is most often very similar to the name of the image file that it is describing.

Once you have found the image, assign it to the Image field in the option you created in the Knowledge Check Task.

Text

To find the text for an option , refer to Labs 6-10 Lab Procedures and find the knowledge check step in the table related to the lab you are working on. Copy and paste the option text into the Text field for the option.

Feedback Text

Follow the procedure outlined above except using the feedback text for either correct or incorrect.

IsCorrectAnswer

Use to mark which option(s) are correct. If the knowledge check is a prediction, this value is used to determine the type

Prediction Knowledge Check Setup

If you are creating a Prediction knowledge check, first you will need to create a complete KnowledgeCheckTask.

Then, set IsPrediction to true and create a new KnowledgeCheckPredictionFeedbackTask under ScriptableObjects → Tasks → KnowledgeCheckPredictionFeedbackTask. Assign your chosen knowledge check in the Chosen Task field.

For context, when you select an option during a knowledge check in the simulation, all of its values are assigned to the selected option fields in the KnowledgeCheckTask scriptable object. Then, when you reach a knowledge check prediction feedback task, it checks the values of the selected option in the chosen task assigned to populate the feedback panel.

Finalization

When all is said and done, assign your completed tasks into their correct positions within the associated lab container (when possible)

Quick Workflow

  1. Create KnowledgeCheckTask

  2. Assign Question + Feedback text and audio

  3. Create and assign option values Labs 6-10 Lab Procedures

  4. If its a prediction knowledge check, create KnowledgeCheckPredictionFeedbackTask and assign it to your created knowledge check

  5. Set IsPrediction to true in your knowledge check task scriptable object
