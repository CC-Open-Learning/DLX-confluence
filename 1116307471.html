VARLab : Bodyswaps End-of-Semester Audit/Clean-up

Created by User cb87c, last modified on Dec 09, 2024

Purpose: At the end of each semester it is important to remove access for the users that shouldn’t have access anymore. As well as deleting classes that are no longer active. Before deleting users and classes, the data must be exported and approved.

Table of Contents

  • Export User data into a CSV file
  • Export Class data into a CSV file
  • Export Summary of Platform Usage
  • Approval Process
  • Deleting Users
  • Deleting Classes

Export User data into a CSV file

  • Login to Bodyswaps

  • Navigate to Users

  • Click on the Menu and click Export Users. This will download a CSV file of all the users.

  • Save the export in the Bodyswaps Microsoft Teams channel Files.

    • Navigate to the Bodyswaps Teams Channel/Files/Bodyswaps Exported Data/Create OR select the correct folder for the term. For example, if you are exporting data for the Spring 2024 term, you would save the file in the S24 folder.

Export Class data into a CSV file

  • Login to Bodyswaps

  • Navigate to Classes

  • Click on the class from which you need to download the data.

  • Click on the Menu and click Download Class. This will download a CSV file of this specific class data.

  • Save the export in the Bodyswaps Microsoft Teams Channel Files.

    • Navigate to the Bodyswaps Teams Channel/Files/Bodyswaps Exported Data/Create OR select the correct folder for the term. For example, if you are exporting data for the Spring 2024 term, you would save the file(s) in the S24 folder.

  • Repeat these steps for each class you need to download the data of.

Export Summary of Platform Usage

  • Login to Bodyswaps

  • Navigate to Dashboard

  • Select the term “Date range”

    • Select the first date of the term in the calendar and then navigate to the last day of the term using the arrows, select “OK” once the two dates are selected.

  • Download the Report (By selecting the Download Green Button) and once it’s downloaded locate the PDF file in your download folders

  • Save the export in the Bodyswaps Microsoft Teams Channel Files.

    • Navigate to the Bodyswaps Teams Channel/Files/Bodyswaps Exported Data/Create OR select the correct folder for the term. For example, if you are exporting data for the Spring 2024 term, you would save the file(s) in the S24 folder.

Approval Process

  • Before deleting users and/or classes, it must be approved by: Corey Berry

  • Request via email manual exports for the term from Bodyswaps Contact (At the moment they can pull more data than we can)

  • Prep exports for approval.

    • Once the user export is ready, create a column with the access status. Mark which users are to be kept and which are to be deleted.

      • Access Status Options:

        • “Delete”

        • “Keep” (If you know the reason why they still need access, note that here. I.E. Keep Admin, Keep Professor)

  • Since it is difficult to remember every user that needs or does not need access, email all the users who are planned to be deleted. This should grab the attention of anyone who may still need access. BCC all users in a single email to maintain privacy and efficiency.

Email Template:

Subject Line: Bodyswaps Account Deactivation Notification

Hello,

We hope you had a successful term and gained new skills using Bodyswaps.

As we have reached the end of the term, it is time to deactivate your account. Your account will be deactivated in two business days.  

If you still need access beyond this date, please reply to this email with the reason for your request.

 

Sincerely,
Bodyswaps IT Support Team

  • Once the deadline is up, delete the classes and the users

Deleting Users

  • Navigate to Users.

  • Ensure you export the user data before deleting the user.

  • Click on the Trash Can Delete Button next to the user that needs to be deleted.

Deleting Classes

  • Navigate to Classes

  • Ensure you export the Class data before deleting the Class.

  • Click on the Trash Can Delete Button next to the Class that needs to be deleted.

  • A pop-up will appear: “Are you sure you want to delete the class named: “…“? Click YES.

The Bodyswaps end-of-semester audit/clean-up is now complete!
