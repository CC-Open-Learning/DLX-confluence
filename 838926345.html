This space provides an overview of all systems currently in place for the Ambulance Orientation DLX. It contains loose class diagrams (They do not follow all expectations of class diagrams). These diagrams are a higher level overview of how many scripts/systems function together. This space is intended to help new and/or returning team members get up to speed on the project quicker, and assist current developers when designing a new system. This is not meant to be a comprehensive documentation of the code base. Clone the repository and read in code comments for a more detailed view

Tools to use:

  • LucidCharts

  • Visual Studio Generated Class Diagrams

  • Figma
