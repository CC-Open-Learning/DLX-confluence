VARLab : Setup Environment to run SAR on Vive Focus 3 and Meta Quest 3: Essential Setup Steps

Created by Nilupul Vithanage, last modified on Feb 27, 2024

Description

This guide provides step-by-step instructions for setting up the PC environment to run SAR (Search and Rescue) application, initially designed for PC VR headsets like Vive Cosmos on standalone VR headsets like Meta Quest and Vive Focus through SteamVR. A high-speed Type A to Type C cable is required to connect the VR headset and the PC to have a seamless execution of VR applications.

What is SteamVR

It is a software platform designed to support virtual reality (VR) experiences on PC. It is a hub for VR content distribution, management of VR devices, and provides essential tools for developers to create VR applications.

Some Key features of SteamVR:

  1. VR Content Distribution: SteamVR offers a vast library of VR games, experiences, and applications available for purchase or download through the Steam store.

  2. Device Management: It supports a variety of VR hardware devices, including headsets like HTC Vive, Oculus Rift, Valve Index, Windows Mixed Reality headsets, and others. SteamVR helps manage the setup, calibration, and firmware updates for these devices.

  3. VR Environment: SteamVR provides a virtual environment known as the "SteamVR Home," where users can access their VR library, interact with friends, customize their space, and launch VR applications.

  4. Integration with Steam Features: SteamVR seamlessly integrates with existing Steam features such as multiplayer matchmaking, friends list, achievements, and cloud saving.

  5. Developer Tools: SteamVR offers development tools, APIs, and documentation to assist developers in creating VR content for the Steam platform.

Set up environment for Meta Quest 3

Step 1: Install all the software.

Step 2: Set up Meta Quest 3 https://www.meta.com/help/quest/articles/getting-started/getting-started-with-quest-3/set-up-quest-3/

Step 3: Connect the PC and Meta quest 3 using high speed cable.

Step 4: Enable Quest link and Developer mode on the device.

Step 5: Allow apps from unknown sources in the Oculus Application > Settings > General.

Step 6: Run SteamVR

Step 7: Run SAR application.
