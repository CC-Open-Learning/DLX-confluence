Before Beginning

This section doesn't cover keyframe animations. Please refer to Unity documentation on how to create them. https://learn.unity.com/tutorial/creating-keyframed-animation-in-timeline

Don’t forget to add Occlusion Portal to your keyframes, or else the door will open and the other side won’t be rendered

For details on animator controllers, please refer to Animator Controllers Setup

Please check that a prefab for the door you want does not already exist. These can be found at “Assets/Prefabs/Doors And Doorways”

Setup the Back Kitchen Doors Behaviour

In this section, you'll find the steps to set up all doors in the back Kitchen. The setup for the other restaurant areas will be similar.

If you have an inspectable attached to the door, please also refer to: Edge Case: Loading Image For Inspectable Door

Single Direction Door

1. Add an animator controller to the walk-in door. Try to make sure this is in the prefab if possible, and add the correct controller.

see Animator Controllers Setup on how to set up controllers.

2. Add the trigger volume prefab which should be located at “Assets/Prefabs/TriggerVolume”.

2.1 adjust the collider to fit over the way point

3. Set the Animator, Action and State Parameter. If the door opens in a single direction ignore everything from “Is Double Door” and “From Inside” fields.

3.1 Select or Drag the door game object for the Animator property.

3.2 Set Actions according to the Header name and State Parameter to "DoorOpen"

4. Add the DoorAudioSource prefab found at “Assets/Prefabs/Doors and Doorways/DoorAudioSource”. Move it to where you want to hear the audio come from (usually the centre of the door or hinges is fine).

5. Add the Door Audio(script) as a component to the Door. Link the Audio Source to the source field and add audio clips to the sound fields. Audio clips can be found at “Assets/Audio/SoundEffects”.

Double Directional Doors (Double Doors)

Since double doors open in different directions depending on where the player is moving, the setup has a few extra properties.

1. Set-up is the same as the single door except the TargetDoorAnimBehaviour script.

2. Check the “Is Double Door” check box and set the “From Inside Fields”.

3. That covers how to set up animated doors!
