VARLab : TE UI Creation from Figma Documentation

Created by User 1c73e on May 23, 2024

This document will outline the steps needed to create UI components within Unity as specified in the Figma documentation below:

https://www.figma.com/design/bjRTVV086WFMxQmPz1MbUQ/Trades-Electrical?node-id=664-692

  1. Find the UI component within the Figma document you wish to add to the Unity project. For this example, we will add the Confirmation message in the Measurement section. On the right side, the Properties panel contains the details of the component. This will be used to recreate the container within Unity:

  2. To export individual components, such as the info icon, select Export in the top right corner after highlighting the desired component. Click the 1x dropdown and change to 2x for larger and 4x for smaller components (this can be scaled down as needed within Unity). Export as PNG :

  3. Drag the .png file you exported into Assets/Art/Sprites/UI/Icons. Change the Texture Type to Sprite (2D/UI). Change the max size to 64. Save.

  4. For each new UXML template needed, create a folder within UI Toolkit name appropriately:

  5. Within the folder, right-click and navigate to Create > UI Toolkit > Editor window:

  6. You will be prompted with the following:

    Uncheck USS (unless ticket explicitly asks for one). Name the UXML file appropriately, and name the C# the same with the suffix Controller

  7. Move the .cs file to Assets/Scripts/UI

  8. Go back to your new folder and open the new UXML template.

  9. Start with a Visual Element that will contain all the individual components. This element positioning will always be Absolute while containing elements will be relative:

  10. Refer back to the Properties section of the Figma documentation to configure the components. The font style and colours will always be 1-1, but the other settings may need adjusting:

  11. To add the icon, drag the icon from Assets/Art/Sprites/UI/Icons into the background slot of the desired Visual Element
