What is a Milling Machine and what is it used for?

The milling machine uses rotary cutters to remove material by advancing the cutter into the stationary work piece using one or more of the table’s 3 axes​.

Major Parts

  • E-Stop

  • Main Power Switch

  • Brake

  • Power Feed Control

  • Hi/Low Gear Selector

    • Only to be used when spindle motor is not running. Make sure selector is in gear before turning spindle on.

  • Spindle

  • Spindle RPM Control

    • Only to be used when spindle motor is running.

  • Spindle On/Off

  • Tool In/Out

Main Hazards

  • Loose clothing, loose hair and jewellery

    • Never operate any machine with anything dangling from your body, such as drawstrings on a sweater or pants, long hair, bracelets, etc.

  • Sharp cutting tools

    • Always grip sharp cutting tools with a cloth or a rag.

    • Or, grip the tool as shown in this reference video.

    • Never pull long chips off a tool with your bare hands, use pliers instead.

  • Improperly clamped work pieces

    • Always ensure the part is properly secured in the vise, as shown in these reference photos:

      • Correct

      • Incorrect

  • Flying chips

    • Always wear safety glasses, and do not wear them at the tip of your nose. This increases the chances of chips contacting your eyes​.

    • Adjust cutter position and direction of cut to control the direction of flying chips.

  • Rotating cutter

    • Never touch a rotating cutter, even at low speeds.

  • Debris on and around the work space

    • Make sure there are no metal chips or loose objects such as tools or rags on or around the machine before you start your work.

Starting up the machine safely

  • When turning on the main power, make sure the E-stop is deactivated, otherwise the machine will not turn on.

  • When inserting the tool using the tool in/out controls, be sure to hold the tool correctly to avoid cutting your hands.

  • While placing the work piece into the clamp on the workbench, position it properly so that it is held tightly, with no gaps between the piece and the clamp.

  • Sometimes, when you change gears before starting your work, you might have to readjust the tool.

  • Make sure your work space is clean before starting.

Turning off the machine safely

There are different ways to safely turn off the Milling Machine.

  1. Turning off the spindle will not bring the spindle to an immediate stop. It will coast to a stop.

  2. The brake can be used to bring the spindle to an immediate stop. However, the break does not disengage the drive motor, and can only be used once the spindle motor has been turned off.

  3. Using the E-stop will also only cut power, and the spindle will continue to rotate and coast to a stop.  To stop the machine spindle from rotating, you must apply the break after the E-stop has been pressed. Use the E-stop only if there is “danger of life or limb.”

When you are done with the machine, be sure to clean off any scraps or metal chips.
