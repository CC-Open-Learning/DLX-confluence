VARLab : Visual Editing for the Damage States

Created by Rounuk Kumaran (Unlicensed) on Aug 22, 2022

We will be mostly utilizing the Images and Gifs to showcase our damaged states, especially ones we cannot represent using videos.

Though there can be various methods of having Images edited to have damage or changes shown for Image-based or Gif-based damage, the following will show some easy steps to create such edits on images. These steps will give you in brief an understanding of what we require and how to achieve it.

Images:

Software used: Photopea (Free Software/ similar to photoshop in UI and use)

We Use the Layers System in the Photopea for this transition.

We have multiple steps for a good damaged state, basically split in three different layers going from bottom to top (layer 1- layer3)

  • Layer01- Base image

  • Layer02- Damage alpha (Cracks, leaks, corrosion, etc.)

  • Layer03- Color-correction or Composition layers that help blend the two layers.

One of the most crucial elements to use is the blending mode.

Gifs:

In similar aspects, we can have Gif Images that can be made using a base image and a couple of frames of animated damage.

The first thing we need is animated Gifs to go on top of the Base images (mostly good states of inspectable assets that we have photographed).

These Gifs available online have either a transparent or opaque background. What we require for the gif to overlay on the base image is one with transparent background. We can convert most Gif clips from having an opaque background to a transparent background using the https://www.online-image-editor.com/ online free editor.

A brief introduction to using can be found in this video: https://www.youtube.com/watch?v=Jr6vDzfwrKI Most Images online has opaque black background, so it is convenient to use this app to get the foreground separated from the background while making the background transparent.

This now allows us to overlay on top of the base image, any gif image sequence that shows damages, sparks, smokes, fluid leaks, etc.
