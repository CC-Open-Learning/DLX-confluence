Before Beginning

This section doesn't cover keyframe animations. Please refer to Unity documentation on how to create them. https://learn.unity.com/tutorial/creating-keyframed-animation-in-timeline

Currently, we have Double Doors and Cooler/Freezer Doors animator controllers ready to use at /Assets/AnimatorControllers and animations at /Assets/Art/Animations.

Double Door Controller

Parameters

Double doors must rotate towards the next room, so we created 4 Trigger parameters.

OpenFromInside<Trigger>: Door rotates on the negative Y axis.

CloseFromInside<Trigger>: Door rotates back to Y = 0.

OpenFromOutside<Trigger>: Door rotates on the positive Y axis.

CloseFromOutside<Trigger>: Door rotates back to Y = 0.

DoorOpen<Bool>: Control the current door state, where true == open.

Transition properties.

Single Door Controller

Parameters

Open<Trigger>: The door opens towards the current room.

Close<Trigger>: Door closes.

DoorOpen<Bool>: Control the current door state, where true == open.
