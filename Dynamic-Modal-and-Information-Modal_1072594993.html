Dynamic Modal

Creating the Dynamic Modal Scriptable Object

To create a new Dynamic Modal you should first create the Modal Pop-Up Scriptable object by going to Assets/Scriptable Objects/Modal Popups and right-clicking inside the folder and selecting Create>ScriptableObjects>ModalPopup and giving the name of the pop-up to the scriptable object.

Setting Up the Scriptable Object

Fill out the fields in the scriptable object with the text that should be displayed in the dynamic modal.

  • Modal Name: is not displayed and is usually the same as the file name.

  • Modal Title: is the Window Title in the header.

  • Content Title: optional title text that can be added at the top of the content box.

  • Content: information text inside the content box

  • Footer Text: optional text that can be added to the window footer. Itâ€™s default color is blue.

  • Footer Icon: optional icon that can be added to the left of the footer text.

  • Confirmation Toggle Text: optional text for a confirmation checkbox.

  • Primary Btn Text: text displayed on the primary button.

  • Secondary Btn Text: text displayed on the secondary button.

  • Primary Btn Color, Secondary Btn Color, Primary Btn Txt Color, Secondary Btn Txt Color: sets the color for the buttons and the text on the buttons.

  • Btn Font Size: sets the font size for the button text.

  • Alignment: sets the alignment using https://docs.unity3d.com/2023.2/Documentation/Manual/UIElements.html alignment for the different elements listed.

  • Options: toggles of what should be displayed in the dynamic modal, if the window will be displayed over the inspection window or the review window, and if the background should be dimmed.

  • Button Custom USS Classes: custom button classes can be added.

How to Display a Dynamic Modal

To set the content and display the Dynamic Modal call the function HandleDisplayModal( ) and pass the scriptable object as a parameter from the ModalPopupBuilder.cs.

Changing the Buttons and checkmark Callbacks

Before calling the function HandleDisplayModal( ) in the same method, set the button actions for the scriptable object.

 {scriptableObjectName}.SetPrimaryAction(() => MethodCalledByTheButton());
 {scriptableObjectName}.SetSecondaryAction(() => MethodCalledByTheButton());
 {scriptableObjectName}.SetToggleAction((val) => ToggleValue = val);

Information Modal

The information modal has a fixed size and only one button. The image in the message body is optional.

Creating the Information Modal Scriptable Object

To create a new Information Modal you should first create the Modal Pop-Up Scriptable object by going to Assets/Scriptable Objects/InformationModal and right-clicking inside the folder and selecting Create>ScriptableObjects>InformationModal and giving the name of the pop-up to the scriptable object.

Setting Up the Scriptable Object

Fill out the fields with the information that will be displayed in the Information Modal

  • Modal Name: is not displayed and is usually the same as the file name.

  • Window Title: is the window title in the header.

  • Body Title: optional message title located in the message body.

  • Body Message: information message that will be displayed.

  • Button Text: text that will be displayed on the button.

  • Is Title Enabled: the body title is optional and needs to be checked if the body has a title.

  • Is Icon Enabled: the icon is optional and this needs to be checked if the message has an icon.

  • Keep Blur On: if this option is checked everything that is not UI will be blurred in the background.

How to Display a Information Modal

To set the content and display the Information Modal call the function HandleDisplayModal( ) and pass the scriptable object as a parameter from the InformationModalBuilder.cs.
