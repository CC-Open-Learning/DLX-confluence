VARLab : Interactive Knowledge Checkpoints

Created by David Inglis (Unlicensed), last modified on Nov 08, 2022

Knowledge Checkpoints will be inserted at the end of certain sections in the learning experience and will present the learner with interactive activities featuring the ‘drag and drop’ and the ‘scratch card’ functionalities.

These Knowledge Checkpoints will be built on top of the Inspection Element class so that they can be included in the Inspection Manifest which defines the flow of the learning experience.​

Clean the Windshield (Scratch Card)

The Clean the Windshield Knowledge Checkpoint uses the Scratch Card plugin to require the learner to clean away dirt on the windshield of the bulldozer. The learner will be prompted to click on the windshield and a UI panel will be shown with a dirty windshield and instructions for ‘cleaning’ the windshield. Once the window is cleaned past some threshold (75%, 80% approx) then the task will be considered complete.

Documentation and the package files for the Scratch Card plugin are stored in the VARLab Sharepoint.
