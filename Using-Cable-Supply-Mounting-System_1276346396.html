VARLab : Using Cable Supply Mounting System

Created by User c4ba3, last modified on Aug 13, 2024

This document contains information on how to use the cable supply mounting system by setting up a task so to start the cable mounting workflow. This document does not contain information on creating cables using the spline tool.

  1. Modify Mountable Name Script

    1. Open the MountableName.cs Script. Add a descriptive name to the enum to identify the supply cable. After that, add a Description Attribute that is the same as the name you added.

      This example will showcase adding 3

  2. Create a Task Camera

    1. To preview the cable within the system, you must create an object with a cinamachine camera and add related scripts.

      1. Create an empty object with a descriptive name and add CinamachineVirtualCamera.

      2. Add Cinemachine Pan Settings and Zoom Settings

  3. Add Mountable.cs to the supply cable that will mounted.

  4. Modify Task.cs

    1. Go into Task.cs. Add a descriptive name and a Description Attribute for the task you are mounting the cable for.

  5. Setting up Cable Mounting Task

    1. Create an empty object and add the Cable Mounting Task Script

    2. Configure the fields based on earlier steps where the Task Preview camera was added(step 2), Task Name(Step 4), Cable mountable options, which would be supply cables, and then the correct one.

  6. Trigger Mounting System

    1. Now you have to pass the mountable name toTaskHandler.onMountableSelected and trigger the event TaskHandler.OnMountingTaskClick with the task name

Now, cables will be mounted when the events in step 6 are invoked with the camera view that is set up.
