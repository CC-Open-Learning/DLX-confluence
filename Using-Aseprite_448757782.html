VARLab : Using Aseprite

Created by Hannah Schmitting (Unlicensed), last modified on Sept 30, 2021

This group of documents outlines the basics of using Aseprite.

  • Creating a New Canvas
  • Preview and Basic Tools
  • Grid Settings

Creating a New Canvas

  • Open Aseprite

  • Go to File > New… or use the keyboard shortcut Ctrl+N.

  • Set width and height, select colour mode (RGBA for Campus RPG), and background.

Preview and Basic Tools

Once you’ve created your canvas, you can set up a preview window to view a specific sprite size as you draw.

  • View > Preview or F7

Over to the left is your colour bar. At the top is a default palette, and colours that are added to your canvas will be added to this palette. Below this is colour selection and alpha bar. You can change the transparency of any colour you pick by sliding the black circle left or right. Under this is the colours that are currently selected. To select new colours, left click to change the primary colour and right click to change the secondary.

To the right is your tool kit. Most of the basic drawing tools you’ll need are here. These will be covered in a separate document.

Grid Settings

If you have a large canvas for a tile set or a larger drawing that you’d like to visually sectioned off into tiles, go to

  • View > Grid > Grid Settings

  • Input width and height preferences and press Okay

  • (RPG Maker MZ tiles are 48x48 pixels)

If you would like to see a grid of individual pixels, go to

  • View > Show > Pixel Grid or use the shortcut Ctrl+Shift+' [Apostrophe]
