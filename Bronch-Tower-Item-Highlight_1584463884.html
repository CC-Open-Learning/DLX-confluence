VARLab : Bronch Tower Item Highlight

Created by Nilupul Vithanage, last modified on Jan 29, 2025

This document provides detailed instructions on how to add an item highlighter to the bronchoscopy tower items. The highlighter will visually emphasize selected items, enhancing the user experience by making it easier to identify and interact with individual components. It includes step-by-step guidance on implementing the highlighter feature

Steps

Open the Bronchoscopy Tower Prefab.

Navigate to the Bronchtower prefab in your Unity project. You can open it in two ways:

  • Project Tab: Locate the prefab in the Project tab and double-click it to open.

  • Hierarchy: Expand the Bronchtower prefab in the Hierarchy by clicking the arrow next to it.

Select the Target Item.

Inside the prefab, identify the item you want to add the highlight to. Click on the specific item to select it.

Add the Highlight Component

With the item selected, add the BronchTowerItemHighlighter component to it. You can do this by:

  • Clicking the Add Component button in the Inspector.

  • Searching for BronchTowerItemHighlighter and selecting it from the list.

Configure the Highlighter Script

In the BronchTowerItemHighlighter script, you will need to configure the following:

  • Item to Highlight: Assign the item you want to highlight by dragging it into the designated field in the script's properties.

  • Transparent Material: Locate the transparent material in the Materials folder. Use the file named Bronch Tower Transparent.mat and assign it to the appropriate field in the script.

Setting Up Explorable Events

Expand the Explorable object associated with the bronchoscopy tower. You will see two events:

  • OnHighlightExplorable

  • OnUnhighlightExplorable

Link the Highlighter Component:

  • Drag the object with the BronchTowerItemHighlighter component into these events.

Assign Highlighter Methods

  • For On Highlight Explorable:
    Assign the BronchTowerItemHighlighter.HighlightExplorable method.

  • For On Unhighlight Explorable:
    Assign the BronchTowerItemHighlighter.RestoreOriginalMaterial method.

Setting Up POI

  • Select the Relevant POI.

Identify and select the relevant Point of Interest (POI) from your scene or prefab.

  • Add the POI to the Explorable List.

In the Explorable object, add the selected POI to the Explorable List

Setting up the Event System

Add the item to the Event System to enable highlight effects. This will ensure the item highlights when the mouse enters and returns to normal when the mouse exits.

Now it’s all set make sure you update and save the GUID manager before you run the simulation.
