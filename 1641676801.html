Settings

  • Verify Settings dialog is displayed when Settings menu option is clicked.

  • Text is clear and readable, spelling and grammar are correct

  • Sound - this is an on/off toggle. Default state is ON. Please note that if you turn off sound, the “Play to Listen” button will not make any sound when clicked. Feel free to test this, but it is working as intended if there is no audio at all when Sound is toggled off.

  • Volume - this is a slider. Default state is full

  • Camera Sensitivity - another slider. Default state is fast (full slider) Note that this setting applies only to manipulating items in the Object Viewer, it does not affect the speed of automated camera movement when entering or exiting POIs.

  • Adjust settings as desired and remember your selections.

  • Click X to close Settings - dialog closes on demand.

  • You can change your settings at any point while playing, it’s not important to set everything perfectly on the first try.
