Demo Outcomes

A recording of the demo has been saved to Microsoft Stream and is accessible to all VARLab team members.

  • Inspect the weld line down the vertical side of the jack stand where the line in currently - this is most likely point of failure if the weld is to fail
  • The jack stand should make a clunky ratcheting noise when moving up if the automatic locking mechanism is working. If the lock is seized or not working, it will not make this noise
  • If the lock on the jack stand is working, the jack will stay in position when the user lets go of it. This does not make any sound
  • If the lock on the jack stand is NOT working, the jack will fall when the user lets go of it. This will make an audible thunk
  • We need to uplift the intro video a little. Kevin raised some good points about making sure that we're covering all the ways that a user can interact with objects in the sim.
    • How to show/hide the checklist
    • How to interact with items to see if they are loose/secure
