VARLab : 1. General Information (UI and functionality)

Created by Colm Ambler (Unlicensed) on Sept 08, 2021

RPG Maker MZ is a fairly simple to understand engine especially if you’ve used most other game engines before.

This is pretty much what you’ll be greeted with (aside from the Hello, that was all me.)

1.1 Menu Bar

First, in the top left we have the basic menu board that will soon be obsolete as right below it are the same commands.

We can access other projects we have downloaded, create new projects and deploy (build) our game through file, and that’s the most important part here.

1.2 Common Tools

Right below the menu bar is another that holds easier access to all of our common tools we’ll be using. We’ll be going through them in chunks as many of them relate to each other.

File

First, from left to right:

New Project: This creates a new project, it’ll ask you for a name and file destination.

Open Project: This will open a project you have downloaded.

Save: This will save your current project.

Undo

Undo: This undoes the last action you took. This can also be done with ctrl z.

Modes

From left to right:

Map Mode: This is the mode you’ll need to be in when you’re creating levels (known as maps in RPG Maker).

Event Mode: This mode allows you to create events which sort of act as functions in the game that can be called in a variety of ways.

Layers

Each of these represent layers within map mode. They’re represented visually but the furthest left picks the layer automatically when you place the tile.

Drawing Options

These are simple drawing options usable while in map mode. From left to right:

Pencil: Draws a tile where your mouse is.

Rectangle: Drag and make a square.

Ellipse: Drag and make a ellipse.

Bucket: Paints all the relative tiles of a tile you select at once.

Shadow Pen: Draws shadows where you put your mouse.

Scale Options

These move the field of view closer and further away. 1:1 displays the map at 1:1 scale. These can also be done by holding ctrl and scrolling the middle mouse button.

Database and Plugins

From left to right:

Database: This holds information about the game such as the characters, their classes, abilities, items, title screen, etc.

Plugins: This is the tab that you use to manage the plugins you’re using.

Functionality

From left to right:

Sound Test: This allows you to view all the sounds you have within the project and listen to them at varied pitches and volumes.

Event Search: This allows you to search for events within your project.

Resource Manager: This manages all the assets you have within the project. This is also where you add external assets in.

Character Generator: RPG Maker’s easy to understand character generator with many different options to allow for varied characters.

Play: Plays the game

Map Tileset

This is where you pick the tiles you want to paint onto your map. You must be in map mode, otherwise you’ll see a list of events.

When you create a map you’ll need to input which tileset you’ll be using. Each tileset can hold multiple tilesheets which can be switched between using the ABCDE tabs at the bottom. Note: Only one tileset can be used per map.

Map Index

This section will list all the maps you have within your project. You can folder them into each other but other organization is quite limited.

Map View

Where most of your attention will be placed. This is where you’ll paint your maps and place your events. In this example, I’ve used a starry night tole as the background with a grass autotile to make a message on a 21X12 map.
