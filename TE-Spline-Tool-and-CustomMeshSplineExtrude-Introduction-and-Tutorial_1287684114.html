VARLab : TE Spline Tool and CustomMeshSplineExtrude Introduction and Tutorial

Created by User 1c73e, last modified on Aug 16, 2024

This page will introduce the Unity Spline tool and outline how to use our CustomMeshExtrude tool in the trades electrical DLX.

Please visit this link for an introduction on Unity Spline https://docs.unity3d.com/Packages/com.unity.splines@1.0/manual/getting-started-with-splines.html

How to add CustomMeshSplineExtrude within TE DLX

  1. Create a new game object, and add a Spline component:

  2. Press the + on the Spline component to add Knots. You will then see a line in the scene:

  3. If you click on the Spline view in the Toolbar as shown, you will then see the knots and the tangents. You can also use the View dropdown to edit what the Spline view looks like:

  4. For this project, after adding the Spline component, then add the Spline Extrude component to the game object as well, and configure as shown:

    Note* these are base values for our wire system, please adjust according to your use case

  5. To edit the Spline, use the move tool on individual Knots to move it as shown here:

    OR you can move the tangents and adjust the spline as shown here:

  6. Make sure all Knots are set to Auto in the Element Inspector:

  7. Add the Custom Mesh Spline Extrude component as shown, and each time you make adjustments to your spline, click both regenerate buttons as shown:
