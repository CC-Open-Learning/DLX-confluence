VARLab : Item-Specific Flavour Text

Created by Hannah Schmitting (Unlicensed), last modified on Jul 27, 2021

Trash Bins

>> This is where the trash goes.

>> Why is there a fake mustache in the trash?

>> I’m not sticking my hands in there. … Hands? Wings.

>> I’m sure there would be something useful in here if this were a different kind of game.

>> I don’t want to get my feathers dirty.

Recycling Bins (Blue)

>> Glass bottles, plastic bottles… is that a pickle jar? It’s probably fine.

>> The blue bins are for plastic, metal, and glass.

>> Someone left a a whole plastic lunch box in here.

>> I don’t think I need to touch this.

>> Sometimes people ask me why I’m so interested in recycling bins.

Recycling Bins (White)

>> That soda can should be in the blue bin.

>> That’s an interesting way to fold a pizza box.

>> These white bins are for paper products.

>> Some of the lids on the white bins look different. In real life, I mean.

>> There’s half an essay in here, I hope it’s a rough draft.

Red/Glass Vending Machine

>> Delicious snacks!

>> Oh neat! I can use my OneCard with this if I want to get a snack later.

>> I must resist.

Coca-Cola Vending Machine

>> So many choices…

>> It’s nice that I can use my OneCard with these vending machines.

Red Bull Vending Machine

>> I don’t need a pick-me-up right now, but it’s good to know this is here.

Pepsi Vending Machine

>> What’s the plural of “soda”?

>> Maybe I’ll get a drink later.

ATM

>> I don’t need to withdraw anything right now.

Computer Standing Desks

>> I’m sure these are useful for something, but the developer writing this text hasn’t figured out what that is yet.

Water Fountains

>> There’s nothing quite like a good, cool drink of water.

>> Refreshing!

>> This is so much better than filling up my water bottle at the bathroom sink.

Progress Pride Flag

>> This is the Progress Pride Flag! Daniel Quasar added the arrows in 2018.

>> [Second click text] The black and brown arrows represent LGBTQ+ communities of colour, and the white, pink, and light blue arrows represent the transgender community.
