VARLab : Using itch.io

Created by Hannah Schmitting (Unlicensed), last modified on Mar 28, 2024

This set of documents cover the use of basic features in itch.io , the current hosting site for Conestoga Campus RPG.

 

Main Page

After signing into itch, the top bar of the site will have navigation features as well as a drop-down menu to the side of your profile name. From left to right:

Itch Logo: The itch.io logo will bring you to the main itch page.

Developer Logs: This link will bring you to the latest developer logs (devlogs) of other creators. This is not where you access your own devlogs.

Jams: From here you can access information about game jams you’ve participated in, host your own game jam, and browse current and upcoming game jams hosted by other creators.

Dashboard: This brings you to your creator dashboard. There are a lot of useful links and tools you can access from here. These are covered under the Creator Dashboard portion of this document series.

Feed: This is where you can view recent updates from creators that you follow, as well as global and featured updates.

Community: This is where you can build, participate in, and manage your community experience.

Bell Icon: This brings you to your notifications.

Profile Image and Name: This will bring you to your itch.io profile, where people can access the games and information you’ve made publicly available.

Arrow: Drop-down menu with even more links.
