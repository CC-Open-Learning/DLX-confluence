Date: Monday Dec 18, 2023

Questions to ask CE (Asked Wed Dec 20, Answered Jan 15/24)

Please post your questions here to ask the CE by this Wednesday Dec 20

  1. Could we be provided with a list of code violations that should be reflected in the DLX? Yes, this will take some time. This will also depend on the final structure of the project. Once there is a project layout to look at, it would be easier to run down a list of possible violations.

  2. Could you specify what tasks the students are expected to complete in the Final Inspection that are different than the Rough-In Inspection? Rough-In Inspection – all cables run. Proper boxes mounted correctly. Cables terminated to the box, and the bond conductors are tied into the bond screw.

    Final Inspection- all devices correctly installed, mounted on the box, cover plates on (when appropriate.

  3. Do you envision that the students could start anywhere, meaning with any FOUR of the light box, fan box, gang box or panel, OR do you see it as unlikely that they’d start at the panel, and so we could leave the panel process until AFTER they’ve installed and wired the other THREE? Yes, they could start at any of the devices, and leave the panel wiring to the end.

  4. How the learner should run a wire from one place to another? Should the learner actually walk with the wire, or can they enter values such as wire height from the floor and distance between staples, and it will automatically connect the wire? I think a drag and drop would work best for this. They can cover specific heights and distances in the shop class.

  5. How does the learner have access to the schematic in the real-world? Should the learner be provided the schematic through on-demand access on the screen? Or lying on a table for them to perform an action? The project directions are provided via eConestoga. They need to complete the schematic based on the lesson the week prior, the required questions to be answered, and the project directions/description.

  6. It was mentioned that Homeowners may not comment on the build until after the FINAL inspection? Would the Red Seal not correct an apprentice’s work if it gets that far? There is a difference between a legal installation and providing the customer what they want. You can install equipment legally in the wrong place.

  7. If we give the students a Code # that they have violated in their Inspection report, will they be able to look up that code number in their textbook based on the Code number, or will we need to give a page number too? Inspection Deficiencies are given to Contractors as the basic Code #. They will still have to read through the rule and determine which part of the rule the violation is referring to.
