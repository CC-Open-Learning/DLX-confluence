VARLab : Character creation guide

Created by Max Verspaget (Unlicensed), last modified on Jun 18, 2021

Character Design

  1. Decide on what type of character this will be (e.g. Condor Mascot)

  2. Gather reference material, as much as necessary

2. Sketch out character design, maintaining proportions with skeleton as base (3:1 head to body ratio)

3. Get creative!

4. Sketch turnarounds

Sprite Creation

  1. Open Sprite sheet template Aseprite file

Stock sprites layer is just for reference, use it and the guide layer for shape of sprite

NOTE: Individual sprites MUST be 48x48px

2. Open character template file

3. (optional) Import concept sketches as reference layers

4. Line character sprite

5. Apply flat colors (over line)

Use reduced opacity line layer

Keep head on separate layer from body

6. Render using stock sprites as style match reference

7. Copy sprite to sheet and replace template sprite by erasing it from the Stock Sprites layer

8. Repeat for all sprites for that character

NOTE: Sprites can be flipped horizontally with Shift+H

9. Once character is completed, export file as a PNG to [project folder]>img>characters, using proper naming convention

Implimentation

  1. In RPG maker MZ, go to Tools>Database

2. In the Actors tab, select to actor you want to apply the sprite to, then double click Character

3. Find your custom character spritesheet and select the proper sprite set

4. DONE!

Character Profile Photo Guide:

  1. Download Template PSD file.

2. Create drawing layer inside the created folder (This will apply the masking to maintain the bounds of the profile)

3. (Optional) Use the Character Generator for REFERENCE if needed

4. Draw character profile (Pay attention to edge shading and line color, use RPG maker stock portraits as color reference)

5. Export to [project folder]>img>faces

6. Impliment following same steps as for walk sprite, selecting Face instead

7. Select correct portrait

8. DONE!
