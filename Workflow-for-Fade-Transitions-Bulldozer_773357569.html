Creating A Fade Transition For The Start Of An Inspection

Step 1 - Confirm assets are setup

Check to ensure that your animations folder has the following files:

Now double-click on the dashboard position animation. This should load the animator window like below:

Next, select the FadeCam node (the orange node) in the animator, and your inspector window should populate with the node details. The inspection should show under the motion row that the FadeCam animation is being used, as shown below:

If the inspector window does not match what is above, you must drag the FadeCam.anim that you added a few steps back into the highlighted area that is shown in the image above.

Step 2 - Setting up the components for the camera position

Now you can close out of the animator and navigate to the camera position where you would like to add the fade transition to the beginning of (A fade transition will only show in the scene when the inspection is moving to the position you select).

After selecting the camera position you would like to add the transition for, you will need to add two new components under the inspection window, a cinemachine storyboard (scripts) and an animator (see image below).

Before continuing, ensure to drag the dashboard position controller into the animator component and drag the fade image png into the cinemachine storyboard script and set all other values to match the values in the screenshot.

Next, you must link the animator component to the CinemachineVirtualCamera inside the inspector window of the same camera position. To do this, click the plus button inside of the yellow highlighting (see image below) and drag the animator component from the bottom of the inspector window into the box under the Runtime Only box. Then ensure the box beside the animator link is checked off and that the dropdown above the checkbox is set to Animator.enabled.

Before continuing to the next step, ensure that all other settings in the above screenshot match the settings of the CinemachineVirtualCamera in your project.

The last step involved with comment setup is to confirm that the other components are set up. These components should already be in place from when the camera position was first implemented. See the image below to confirm the Cinemachine Recomposer, cinemachine zoom settings, and cinemachine pan settings are added to the camera position.

Step 3 - Add the camera transition to the Main Camera

The last step of adding a fade transition is to add it to the main camera of your project.

Under the Hierarchy, look for the main camera like below:

Once you have clicked on the main camera, look in the inspector panel for the cinemachineBrain (as below).

Then you will need to select the plus button on the custom blends area. Which will look like the following:

Under the “From” tab, you will need to expand the row and select **ANY CAMERA** as the image shows below:

And for the “To” tab, you will need to select the camera position you just added the components to (the position you are adding the fade to the beginning of). In our example case, this is the Travel Pedals and Sticks Position. Once you have filled in both the “From” and “To” boxes, change the Style option to cut, as the screenshot below shows:

And that's it! At this point, you should have successfully added a fade transition for the start of your camera position.

Note: Only one image asset and one animation asset is necessary for a single project no matter the number of fade transition in the project
