Each tester should add their name and hardware specs to the Test Team table. Coordinate your efforts with other testers to provide the best possible coverage for browsers, OS, and hardware variables. After executing each test, the tester should add their name (use @ to tag your Confluence account). It is normal for tests to be executed more than once, do not overwrite other tester names.

Task List - Select and Mount Boxes

Click the Task List button on the main toolbar to open the Task List dialog.

  • Verify that you cannot click on tasks that are not available

  • Click on the first available task, Select and Mount Boxes

  • Verify that text and images are clear and readable

  • Click each task tab, verify that each tab shows options that are appropriate for that task and you can do these tasks in any order

  • Click to select a box and then click to select a preview location for that box in the bathroom. Preview is shown.

  • Click Redo. Make sure you can select a different location or the same location you selected the first time. Each time you select a location, you can click Redo to select location again.

  • Click Confirm. Verify that box is mounted at the selected location.

  • Return to the Task Menu, verify that you are automatically taken to the first (leftmost) uncompleted task in the currently In Progress task group.

  • Click to select another box and click to select a preview location for that box. Preview is shown.

  • Click Redo. Make sure you can select a different location or you can select the same location as the previous preview. Verify that you can preview different locations before you Confirm.

  • Click Confirm. Verify that the box is mounted at the selected location.

  • In the Task List, select a box mounting task that has already been completed (shows INSTALLED). Verify that you can select the same box that is already installed or you can select a new box for this task. Once you have selected a box option, verify that you can select a new location or you can select the previously selected location for that box.

  • Click on a box that has been mounted in the scene. In the small task revision box, select either Move Box or Replace Box to make changes. Verify that you can select the box as many times as you want and change your selection each time.

  • Repeat until all boxes have been mounted. Correct answers are shown below, but every tester should make AT LEAST 2 MISTAKES during this task.

FAN BOX - 7-1/4" x 7-1/2" Exhaust Fan Box - Ceiling above toilet/bath

LIGHT BOX - 4 x 1-1/2" Octagonal Box with bracket - Ceiling at room centre

OUTLET BOX - 3x2x3" Device Box - Wall behind vanity

GANG BOX - 2-1/2" 2-Gang Box - Wall beside entry door

Revise Tasks - Select & Mount Boxes

Because you made at least 2 mistakes, you will have to fix them before you can move on.

  • Click on the Task List to fix your first mistake.

  • Verify that you cannot select any other tasks, only Select & Mount Boxes tasks are available

  • Verify that you cannot change any of your correct answers (you can view them)

  • Navigate to one of the tasks you got wrong and fix your mistake (either box or location, or both). Verify that the selected box is mounted at the selected location.

  • Click on Supervisor Feedback again and verify that feedback is up to date (corrected item now shows as correct)

  • Correct your other mistake by clicking on the box in the scene (do begin by clicking on the Task List)

  • A small options dialog is shown, note that you can only change a wrong answer (if location is wrong, you can only Move the box, if the box is wrong, you can only Replace the box)

  • Verify that you cannot confirm your incorrect selection, you must select one of the 3 remaining options.

  • Repeat until all Supervisor Feedback is correct, verify that you get a Great job! message.

Task List - Supply Cables

  • Verify that Select and Mount Boxes task is marked as Complete

  • Verify that Supply Cables task is marked as Available

  • Select Supply Cables, then select subtask Supply Cables to Boxes

  • Knowledge Check - When an incorrect answer is selected, a toast message appears to provide immediate feedback. Verify message text is clear and readable. When a correct answer is selected, you are moved on to the next question or to the first task in this subtask group.

  • Select cable routing options for each task. Make sure that the thumbnail image in the Task List option menu matches the appearance of the preview in the scene. Confirm each selection.

  • At any point during this workflow, exit the DLX and return. Select Continue in the Welcome Back dialog. Verify that your completed tasks were saved correctly, and you are ready to proceed with the next task.

  • Verify that you are not required to repeat the Knowledge Check questions for Supply Cables to Boxes.

  • After completing all Supply Cables to Boxes tasks, Terminate Cables in Boxes task becomes Available. Select it.

  • Knowledge Check - verify that this knowledge check behaves in a predictable and consistent way compared to the previous knowledge check.

  • For each Terminate Cables in Boxes task, you must select a conductor to terminate, and then select a termination option for each conductor. Verify that:

    • You must terminate conductors in correct order. It is not possible to terminate them in the wrong order.

    • Each termination option (yellow orb) has a hover tooltip that is clear and readable

    • You can preview any termination option, then change your mind and preview a different option before confirming.

  • At least once during this part of the workflow, exit the DLX and return, selecting Continue in the Welcome Back dialog. Verify that all of your completed tasks have been saved and restored successfully and you can proceed with your next task.

Offline mode

What happens when you load the DLX

  • disconnect the network after successfully loading the DLX?

    • Cloud Save connection status icon changes to Red Cloud With X. A message is displayed on the screen to notify the user that progress is not being saved.

    • Otherwise, the DLX continues to work and play as normal - no detectable change in workflow, display quality, performance.

  • reconnect to the network while still in the DLX but after some additional progress has been made?

    • Cloud Save connection status icon changes to Green Cloud With Checkmark. A message is displayed on the screen to notify the user that progress is saved successfully.

  • Does analytics keep getting recorded?

    • Analytics events were not affected by disconnecting the internet while playing, then reconnecting it before ending the session.
