Introduction

This article covers the process used in PHI to add audio effects in the Unity scene. This article and process is up to date as of 12/08/2024.

Before you Begin

This article assumes you have a basic working knowledge of navigating the Unity Editor, particularly the Inspector, Hierarchy, Prefabs, and Project tabs.

Set Up

  1. Find the object you would like to have the audio source come from. If there is No object or you would like the audio to come from a specific part of the object create an empty and move it to the correct location. Preferably this should be apart of a prefab.

  2. Add an Audio Source component.

  3. Select your audio clip either found at “Assets/Audio/SoundEffects” or just select the arrow in the Resource field

  4. Select the Sound Effects for the Output field this will tie it to the audio mixer and the volume sliders.

  5. set the minimum and maximum distance to set where the audio can be heard from this can be visualized from the circle appearing around the object. If you need to fine tune it you can use custom volume rolloff to make exact ranges but its better to stick to just messing with the range values.

    Now you should be done setting up the audio. Play test it and make sure everything sounds OK. if not mess around with the min and max ranges to fine tune it.
