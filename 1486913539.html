VARLab : Setting Up Dynamically Mountable Prefabs (Boxes, Devices)

Created by Amila Abeygunasekara (Unlicensed), last modified on Nov 27, 2024

To dynamically instantiate a Mountable, we need a MountLocation in the scene.

Mount Location

Select a MountLocation object while keeping the Handle Position as Pivot and Tool Handle Rotation as Local.

The local Front (Z) axis of MountLocation represents the front of the Mountable to be mounted.

The mount points below will act as anchors to attach the Mountable’s pivot point. This can be configured in MountLocation GameObject.

An example of a Mountable instantiated on the Right mount point is shown below.

Mountable Prefab Setup

Ensure the mesh aligns the front (Z) axis of the parent GameObject to properly align with the MountLocation when instantiated.
