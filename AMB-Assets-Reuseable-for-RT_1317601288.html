CC_VARLab - Asset Files - Medication_Dexamethasone_5mlVial - All Documents (sharepoint.com)

  • The general shape of the bottle has been completed for the Ambulance 5ml Dexamethasone medication bottle. With some squared scale adjustments, it should be easily adaptable as a base for a 20ml bottle required for RT’s uses

CC_VARLab - Asset Files - Medication_Epinephrine_1mg - All Documents (sharepoint.com)

  • Not entirely sure how many mg each Ampule of Epi used by RT contains, but the base 1mg created for AMB should be sufficient as either a out-of-box reuse case, or as a base for modifications to a needed size

CC_VARLab - Asset Files - Syringe_Disposable - All Documents (sharepoint.com)

  • Completed 10cc Model already exists in the AMB files that will meet the requirements of whats listed. Some additional work is needed to modify the existing pieces into what will be needed, including:

    • Separating out model into component parts (plunger and body) → low effort

    • Creating a package mesh that can be rigged and animated for separation

  • Should be fairly quick work to create package similarly to how other packages were completed. Should also require at most 2-3 bones (+root) to enable the package to split and contents to be removed

CC_VARLab - Asset Files - Airways_Nasopharyngeal - All Documents (mcas.ms)

  • A lubrication packet was created for the AMB Nasopharyngeal Airways asset

Asset Files - Forcep_Disposable
Asset Files - Instrument_Medical_Shears
Asset Files - Scissor_Surgical

  • Some instruments were created for the AMB project. Some are likely to be reusable, but will need a confirmation of asset requirements for whats needed

Asset Files - Gauze_2x2

Asset Files - Gauze_4x4

  • Gauze Assets were created for AMB and should be reuseable for RT

CC_VARLab - Asset Files - Bottle_Saline_500ml - All Documents (mcas.ms)

  • A 500ml Bottle of Saline was created for AMB and should be reuseable on RT

  • An asset revision may be required if a 1000ml bottle is required

Asset Files - Mask_BVM_Adult

Asset Files - Mask_BVM_Pediatric

  • A BVM set of Adult, Pedi and Neonatal masks has been completed for AMB and should be reuseable. Some disassembling/reassembling may be needed depending on what the needed interactions are (similar to the 10cc Suction Syringe)

CC_VARLab - Asset Files - SuctionUnit_Compact - All Documents (mcas.ms)

  • Suction tubing was completed for one of AMBs assets and should be reuseable

  • Packaging will need to be created
