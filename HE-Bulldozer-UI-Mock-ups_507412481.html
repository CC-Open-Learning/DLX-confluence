VARLab : HE Bulldozer UI Mock ups

Created by Harshpreet Singh (Unlicensed), last modified on Aug 10, 2022

We have come up with 3 UI design basic ideas of what it can look like. These designs will not be completed replicated as shown below and these can be changed after feedback. There is more to be added in these mock-ups like camera zoom slider, next inspection, checklist and more.

Option #1

This is the very first option for the UI mock-up. We made the UI in a way so it is very easy on the Users and everything will be very easy to read. There will be a button on the inspectable when clicked it will open two small prompts side by side. One of the prompts will be the good states and the other will be the bad states. In this example, one of the bad states is clicked and it opens an overlay with the picture/video of the bad state with the description of it. Same for the good state and other bad states.

9ae91656d1086109d987b759b3052b03.mp4

Option #2

This mock-up has the same Inspectable button same as the option #1 and clicking the button will open a window with good and bad states listed. There will be an Info button on each state option and when clicked it will open a new window that will contain information and pictures/videos of that respective state.

0f6779c34159b0d4638471af5a6aa9f7.mp4

Option #3

In this option basically after clicking the inspectable button further a window will open which will consist of all the good and bad states. Each state will have its own picture/video and also a description under them.
