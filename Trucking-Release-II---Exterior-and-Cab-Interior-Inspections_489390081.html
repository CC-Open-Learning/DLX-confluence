Known Issues - Trucking Release II - as of Friday, May 20, 2022

Jira #

Priority

Description

Impact or Workaround

TI-862

High

Sometimes the cargo van door latches cannot be selected so that I can inspect them.

Occurs rarely, and only if the learner exits suddenly in the middle of inspecting the van doors. To get past this problem, simply click the button to re-open the doors, then click to close them again. Latches can now be inspected as normal.

TI-797

High

Inspection panel sometimes does not appear when highlighted object is clicked

Occurs occasionally. To resolve this problem, click anywhere else on the screen, then back on the object you want to inspect. Panel will appear as normal.

TI-114

High

Suspension inspections (front and rear and trailer) are not yet implemented in Release II

Deferred to Release III to allow for adaptation of scanned 3D models into the scene. This step is skipped in the Release II Guided Inspection.

TI-952

High

Instructors cannot use Save and Load - they do not have a LearnerID from the LMS

No student impact, but this problem is significant anytime there is more than 1 instructor playing the sim in the same course shell. Instructors are not seen as unique, so multiple instructors will overwrite each other’s saved session data, causing unpredictable behaviour.

TI-863

Medium

The PS front fender crack is difficult to see from the inspection point

Learner may need to zoom in to see this clearly

TI-820

Medium

Brake chambers are missing push rod and slack adjuster

Cosmetic, no impact to inspection workflow, but looks wrong

TI-727

Medium

Progress Checklist is not yet implemented in Release II.

This is not very useful in Guided Mode, but will be crucial for Advanced Mode (Release III). This  has no impact in Release II because there is no chance that the learner can skip any steps in the guided inspection.

TI-599

Medium

Results Panel is not yet implemented in Release II.

This is not very useful in Guided Mode, but will be crucial for Advanced Mode (Release III). This  has no impact in Release II because there is no chance that the learner can record incorrect inspection results in the guided inspection.

TI-876

Medium

Sim might not run well on 4GB RAM

Recommend 8GB RAM or higher, we will optimize performance for Release III as much as possible. It may be that this issue cannot be resolved.
