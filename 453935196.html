VARLab : Dev & Art Workflows for Trucking Inspection

Created by David Inglis (Unlicensed) on Oct 28, 2021

Overview

The Trucking Inspection project in Jira has two distinct boards: the Trucking development board which tracks dev work through User Stories, and the Trucking asset board which tracks digital artistry work through a new Asset ticket type which near equivalent to the Story ticket type but is constrained to a new, art-focused workflow.

Shared Backlog

Any TI ticket not in an active sprint will appear in the backlog for both the development and asset boards in the Trucking Inspection project.
